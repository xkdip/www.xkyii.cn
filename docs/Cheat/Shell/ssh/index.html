ssh 备忘

免密​

环境​

  • Client机,可以是Windows系统.
    • 本文以Win10为例.
  • Server机,提供ssh服务.
    • 本文以CentOs8为例.

操作​

  • Server
# 登录为xkyii用户

cd ~
# 如果.ssh目录不存在,就新建一个
mkdir .ssh
  • Client
cd ~
mkdir .ssh

# 生成密钥(连续三次回车,空密码)
ssh-keygen -t rsa

ls
# id_rsa id_rsa.pub

# 把公钥传到Server端(xkyii用户)(输入密码)
scp id_rsa.pub xkyii@Server:/home/xkyii/.ssh/
  • Server
cd ~/.ssh

cat id_rsa.pub >> authorized_keys

# 重启sshd服务(输入密码)
sudo service sshd restart
  • Client
# 此时应该不需要密码了
ssh xkyii@Server

失效处理​

  • 检查AuthorizedKeysFile配置
vi /etc/ssh/sshd_config

# 解开以下注释
RSAAuthentication           yes
PubkeyAuthentication        yes
AuthorizedKeysFile          .ssh/authorized_keys
AuthorizedKeysCommand       none
AuthorizedKeysCommandRunAs  nobody
  • 检查安全日志
sudo cat /var/log/secure

# 如果有以下错误,表示权限不匹配
Authentication refused: bad ownership or modes for file /home/xkyii/.ssh/authorized_keys
  • 解决
cd ~
chmod g-w /home/xkyii
chomd 700 ./.ssh
chmod 600 ./.ssh/authorized_keys
