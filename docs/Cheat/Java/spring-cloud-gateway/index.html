RouteDefinitionWriter 增量更新​

Gateway提供一个路由配置写入接口RouteDefinitionWriter，定义了保存和删除两个方法，代码如下：

org.springframework.cloud.gateway.route.RouteDefinitionWriter
public interface RouteDefinitionWriter {

    /**
     * 保存路由配置
     *
     * @param route 路由配置
     * @return Mono<Void>
     */
    Mono<Void> save(Mono<RouteDefinition> route);

    /**
     * 删除路由配置
     *
     * @param routeId 路由编号
     * @return Mono<Void>
     */
    Mono<Void> delete(Mono<String> routeId);
}

我们可以通过定义一个服务来提供动态刷新路由的功能:

public class DynamicRouteServiceImpl implements ApplicationEventPublisherAware {

    private final RouteDefinitionWriter writer;
    private ApplicationE

    public ApplicationEventPublisherAware(RouteDefinitionWriter writer) {
        this.writer = writer;
    }
}
